# IT TASK 

## What is IT TASK?

is an App for managing IT Assets in the form of Computers, Printers, Projectors, Servers and others. IT Inventory also provides a feature for Open Ticket (Problem Solving Request).

## Framework Used
Codeigniter 4

## Database Used
MySQL
